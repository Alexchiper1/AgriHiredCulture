# AgriHiredCulture

AgriHiredCulture is a Django web application that connects agricultural employers with candidates and supports recruiter review of job applications.

## Live Deployment

The deployed application is available here:

[https://agrihiredculture.onrender.com](https://agrihiredculture.onrender.com)

## Features

- Role-based access for Candidate, Employer, and Recruiter
- Candidate skill management
- Employer job posting
- Job applications with Accept/Decline workflow
